## 描述

基于 react 的 nextjs ssr 服务端渲染脚手架

## 开发

```
yarn
```

```
npm run dev
```

## 部署

```
yarn
```

```
npm run build
```

> nginx 指向 dist 文件

##  启动

```
npm start
```
